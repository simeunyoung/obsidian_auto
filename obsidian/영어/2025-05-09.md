# 주어 확장 Self Check

| 한국어                | 영어                         |
| ------------------ | -------------------------- |
| 너희 엄마 여기 계셔?       | Is your mom here?          |
| 너희 부모님 여기 계셔?      | Are your parents here?     |
| 너희 부모님 여기 안 계셨어.   | Your parents are not here. |
| 너희 엄마 여기 안 계셔.     | Your mom is not here.      |
| 내 책상 여기 있어?        |                            |
| 내 책상 여기 없었어.       |                            |
| 내 책상과 의자 여기 있었어?   |                            |
| 내 책상과 의자 여기 없었어.   |                            |
| 네 친구 공부해?          | Does your friend study?    |
| 네 친구 공부 안 했어.      | Your friend didn't study.  |
| 네 친구들 공부했어?        | Did your friends study?    |
