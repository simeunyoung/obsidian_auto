	- 문제 : IoTCore에서 사물을 삭제 시킬 때는 비활성화가 먼저 진행 되고 5분 이후에 삭제가 가능함
	- 해결 : DSIM에서 삭제 요청이 오면 비활성화 처리 후 일주일 마다 비활성화 된 사물 삭제 이벤트 추가
1. Lambda에 해당 코드 업로드 (Python)
2. EventBridge에서 1주일마다 호출 이벤트 설정
```
import boto3
import logging
from datetime import datetime, timezone, timedelta

# 로깅 설정
logging.basicConfig(level=logging.INFO)
logger = logging.getLogger()

# AWS IoT 클라이언트
iot_client = boto3.client('iot')

def fetch_deprecated_thing_types():
    """사용 중지된 Thing Type 목록 가져오기"""
    try:
        logger.info("Fetching list of deprecated Thing Types...")
        response = iot_client.list_thing_types()
        thing_types = response.get("thingTypes", [])
        
        deprecated_thing_types = []
        for thing_type in thing_types:
            metadata = thing_type.get("thingTypeMetadata", {})
            if metadata.get("deprecated"):
                thing_type_name = thing_type.get("thingTypeName")
                deprecation_date = metadata.get("deprecationDate")
                deprecated_thing_types.append({
                    "name": thing_type_name,
                    "deprecationDate": deprecation_date
                })
        logger.info(f"Found {len(deprecated_thing_types)} deprecated Thing Types.")
        return deprecated_thing_types
    except Exception as e:
        logger.error(f"Error fetching Thing Types: {e}")
        return []

def delete_thing_type(thing_type_name, deprecation_date):
    """Thing Type 삭제 로직"""
    try:
        # `deprecationDate`가 datetime 객체인지 확인
        if isinstance(deprecation_date, str):
            deprecation_date = datetime.strptime(deprecation_date, "%Y-%m-%dT%H:%M:%S.%f%z")
        
        now = datetime.now(timezone.utc)

        # 5분 경과 여부 확인
        if now < deprecation_date + timedelta(minutes=5):
            logger.info(f"Thing Type {thing_type_name} cannot be deleted yet. Wait for 5 minutes after deprecation.")
            return False

        # Thing Type 삭제
        iot_client.delete_thing_type(thingTypeName=thing_type_name)
        logger.info(f"Thing Type {thing_type_name} deleted successfully.")
        return True

    except Exception as e:
        logger.error(f"Failed to delete Thing Type {thing_type_name}: {e}")
        return False

if __name__ == "__main__":
    logger.info("Starting local test for deleting deprecated Thing Types...")

    deprecated_thing_types = fetch_deprecated_thing_types()
    if not deprecated_thing_types:
        logger.info("No deprecated Thing Types found.")
    else:
        for thing_type in deprecated_thing_types:
            thing_type_name = thing_type["name"]
            deprecation_date = thing_type["deprecationDate"]
            logger.info(f"Attempting to delete Thing Type: {thing_type_name}")
            result = delete_thing_type(thing_type_name, deprecation_date)
            logger.info(f"Thing Type {thing_type_name} deletion result: {result}")
```
