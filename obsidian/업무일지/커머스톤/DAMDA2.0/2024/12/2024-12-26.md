- 인수인계 #인수인계
	- 태블로
		- 종류 : 데스크탑(2023), 비주얼라이제이션, 서비스매니저, 프랩
		- 1. 데스크탑(2023)
		- 1.1 DB 주소는 port 51000 에서 작업한다.
		- 1.2 메인 대시보드가 있고 세부 작업은 시트를 만들어 작업 후 대시보드로 모인다.

		- 2. 비주얼라이제이션(작업물 관리)
		- 2.1 작업물 관리(작업 후 작업물 올리는 작업)
			- 메뉴 > 서버(로그인) :29443 > 로그인 정보 입력 > 통합 문서 게시 > 시트(대시보드만)
			- 비주얼라이제이션에서 DB정보 변경 : 데이터 원본 > ... > 연결대상 > 운영 DB로 변경

		- 3. 서비스매니저(라이선스 갱신)
		- 3.1 갱신일 > 박팀장님 전달 > 박팀장님이 연락함 > 새로고침
			- 3.1.1 갱신일 > 박팀장님 전달 > 라이선스 제품키 전달 받음(빅스데이터) > 라이선스 활성화 > 키입력 > 새로고침
			- 3.1.2 갱신일 > 박팀장님 전달 > 사이트에서 직접 제품키 확인 > 라이선스 활성화 > 키입력 > 새로고침
		- 3.2 태블로에서 도움말? > 제품키 관리 > 제품키로 활성화 > 키입력

		- 4. 프랩 (전처리)

		- 포탈의 회원정보에 태블로사용자의 분류코드 == 태블로 프로젝트 이름 일치 해야 권한이 주어짐
		- 권한 라이선스는 admin 외 5개
