
| Verb         | 의미            | 사용 시점            |
| ------------ | ------------- | ---------------- |
| **add**      | 신규 기능 / 신규 코드 | 기능 처음 구현         |
| **update**   | 기능 변경 / 개선    | 기존 로직 수정         |
| **delete**   | 기능/코드 제거      | 더 이상 안 쓰는 코드     |
| **fix**      | 버그 수정         | 의도와 다른 동작        |
| **refactor** | 구조 개선         | 동작 변화 없음         |
| **config**   | 설정 변경         | yml, gradle, env |
| **docs**     | 문서            | README, 주석       |
| **test**     | 테스트           | 테스트 코드           |
