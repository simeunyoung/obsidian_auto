
### 펌웨어 업데이트 구조 설계(AWS)

#### 1. 통신 모듈
 - FreeRTOS OTA 업데이트 작업 생성
#### 2. MCU
- 사용자 지정 작업 생성

MCU OTA는 보통 gateway(통신 모듈)가 대신 업데이트 한다.
```
Cloud
 ↓
ESP32 (gateway)
 ↓
MCU
```

여기서 MCU 전달 방식이 두 가지다.
##### 방식 A
- MCU firmwarae를 S3에서 직접 다운로드
```
IoT Core
 ↓
ESP32
 ↓
S3 firmware download
 ↓
MCU flash
```

##### 방식 B
- ECP32 firmware 안에 MCU firmware 포함
