3. /validate에서 /error, 500 로그가 찍히는데 에러 내용이 안찍히는 현상 해결
	- Controller에서 throws Exception 형태로 예외를 상위로 전달했으나, Global Exception Handler에서 해당 예외를 처리하지 못해, Spring Boot 기본 /error 경로로 처리
	- 예외가 공통 Exception Handler로 정상적으로 매핑되지 않는 구조 문제로 보임
	-> Global Exception Handler에 모든 예외를 처리하도록 확장
    
4. 등록 시 Json null 에러 발생하는데 내용 안찍히는 현상 해결
