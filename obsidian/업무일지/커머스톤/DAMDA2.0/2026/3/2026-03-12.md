전 담당자가 실행한 아래의 명령으로 파일들의 권한이 변경됨.
```
chmod -R 755 ./*
```

git 기준으로 권한 복구
```
git restore .
```

git pull
```
git pull origin release
```

gradlew 실행 권한 부여
```
chmod +x gradlew
```

---

혹은
```
find . -type f -exec chmod 644 {} \;
```


앞으로 파일 권한 변경을 git 변경으로 보지 않도록 하는 설정(적용은 안했음)
```
git config core.fileMode false
```
