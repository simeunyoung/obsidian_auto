

### OTA

기존 FLOW

1. APP -> DCM /ota/start 
2. DCM -> MQTT 발행
3. Device -> MQTT 수신해서 업데이트 진행
4. APP은 구독한 MQTT에서 메세지 수신하면 로딩 완료 -> DCM /ota/check
5. Device -> MQTT 발행
6. Lambda -> iot_device_firmware 상태 변경
7. DCM iot_device_firmware 상태 변경 확인 -> APP response(true)
8. APP에서 OTA update 표시 삭제

1안

1. APP -> DCM /ota/start
2. DCM -> MQTT 발행
3. Device -> MQTT 수신 업데이트 진행
4. APP은 로딩하며 iot_device_firmware에 y/n 상태 변경
5. APP -> DCM /ota/check2
6. APP에서 OTA update 표시 삭제
7. Device -> MQTT 발행
8. Lambda -> iot_device_firmware 상태변경
