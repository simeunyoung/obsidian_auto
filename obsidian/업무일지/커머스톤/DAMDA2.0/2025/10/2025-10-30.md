### 태블로
	2025.11.2 만료 예정 라이선스 업데이트
		개발 서버 업데이트 완료

라이선스 업데이트 중 트러블 슈팅
- TSM WebUI 인증(PAM) 실패로 인한 로그인 불가
		"https://portal.godamda.kr:8850/api/0.5/login?client=webui에 연결할 수 없음"
			원인: 비밀번호 변경일 25.02.24 최대 사용기간 90일로 설정되어 현 시점인 25.10.30 비밀번호 만료
			해결: 비밀번호 변경 및 만료 해제
```
 # L(locked) / PS(정상) 확인
 sudo passwd -S damda
 > damda PS 2025-02-24 0 90 7 -1 (Password set, MD5 crypt.)


sudo chage -M -1 damda   # 비밀번호 유효기간 제한 없음(never)
sudo chage -E -1 damda   # 계정 만료 없음
sudo passwd damda        # 새 비밀번호 지정
```
