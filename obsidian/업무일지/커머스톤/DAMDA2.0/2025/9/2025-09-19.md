AWS
	1. EC2(dsim)에 IAM 역할 연결(정책 및 역할 생성)
	2. EC2(dsim)에 CloudWatch Agent 설치
```
sudo yum install -y amazon-cloudwatch-agent
```
	3. 설정 파일 작성
	   /opt/aws/amazon-cloudwatch-agent/bin/config.json
```
{
  "logs": {
    "logs_collected": {
      "files": {
        "collect_list": [
          {
            "file_path": "/app/DSIM/logs/dsim.log",
            "log_group_name": "DSimLogs",
            "log_stream_name": "{instance_id}"
          }
        ]
      }
    }
  }
}
```
	4.설정 적용 및 에이전트 실행
  ```
sudo /opt/aws/amazon-cloudwatch-agent/bin/amazon-cloudwatch-agent-ctl \
  -a fetch-config \
  -m ec2 \
  -c file:/opt/aws/amazon-cloudwatch-agent/bin/config.json \
  -s
  ```
- 설명
> -a fetch-config
	: Agent에 적용할 설정을 가져오고 반영(fetch)
>-m ec2
>	: 실행되는 환경 모드(`mode`)를 지정
>-c file:/opt/aws/amazon-cloudwatch-agent/bin/config.json
>	: 적용할 설정 파일 경로를 지정
>-s
>	: 설정 적용 후 Agent를 바로 시작(start)

	5. 실행 상태 확인
```
sudo systemctl status amazon-cloudwatch-agent
```
	6. 부팅시 자동 실행
```
sudo systemctl enable amazon-cloudwatch-agent
```
	7. 테스트 메세지
```
echo "테스트 로그입니다 $(date)" >> /app/DSIM/logs/app.log
```
