- [https://lifegoesonme.tistory.com/554](https://lifegoesonme.tistory.com/554) logrotate 설정
	- cat /etc/logrotate.d/dcm 참고해서 모든 서버에 적용
```
vi /etc/logrotate.d/dsim

/app/DSIM/dsim.log {
    copytruncate // 로테이트 수행 후 원본 로그 파일 크기를 0으로 리셋
    daily // 백업 주기 daily, weekly, monthly, yearly
    create 644 damda damda // [권한] [유저] [그룹]
    rotate 30 // 최대 로그 파일 개수
    dateext // 대상 로그 파일명에 확장자 추가 (Default : YYYYMMDD)
    missingok // 대상 로그 파일이 존재하지 않아도 에러 발생 없음
    notifempty // 대상 로그 파일이 비어있으면 로테이트 수행 안함
    nocompress // 로그 파일이 회전될 때 해당 로그 파일을 압축하지 않도록 지정
}

// 권한 설정
chmod 755 /app/DSIM/

// 실행
sudo logrotate -vf /etc/logrotate.d/dsim

```
