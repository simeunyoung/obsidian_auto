API
	UserInfo API 요청 시 2차 인증 (비밀번호) 입력 필요하게 변경완료
	 - 앱에서 로그인 후 userinfo api요청 시 비밀번호 입력을 받을 수 없어 JWT에 필요 정보 리턴하는 방식으로 변경
		 - 홈 이름을 변경하면 jwt에서 가져다 쓰기 때문에 token 재발급 필요
		   홈 이름 편집 시
			 - 기존 로그인 후 userinfo api 삭제
				 - 로그인 후 jwt에서 nickname 사용
			 - 기존 홈 이름 편집 후 userinfo api 삭제
				 - Token refresh 후 nickname 새로 적용
