- tableau 서버 점유율 점검
- 전체 서버에 대해 가용률 점검 진행 예정
	1. 디스크 사용량 전체 확인
	```
	df -h
	```
	2. 대용량 파일/디렉토리 찾기(Top 20 기준)
	```
	du -ah / | sort -rh | head -n 20
	```
	3. /log 디렉토리 내 로그 점검
	```
	du -sh /log/*
	```
	4. 90일 이상된 로그 파일 확인
	```
	find /var/log -type f -mtime +30 -exec ls -lh {} \; | sort -k 5 -rh | head -n 30
	```
	5. 불필요한 로그 삭제(예시 90일 초과 로그)
	```
	find /var/log -type f -mtime +90 -exec rm -f {} \;
	```
