			- startauth.sh 기존의 추출 방식에서 변경
```
# 변경 전
nohup java -jar lib/auth-0.0.1-SNAPSHOT.jar --spring.config.location=config/application-dev.yml,config/application.yml > /dev/null 2>&1 &
echo `ps -ef | grep -v grep | grep auth-0.0.1-SNAPSHOT.jar | awk '{print $2}'` >  /app/portal/oauth/oauth.pid

# 변경 후
nohup java -jar /app/portal/oauth/lib/auth-0.0.1-SNAPSHOT.jar \
--spring.config.location=/app/portal/oauth/config/application-dev.yml,/app/portal/oauth/config/application.yml \
> /app/portal/oauth/logs/oauth.log 2>&1 &

echo $! > /app/portal/oauth/oauth.pid
```
