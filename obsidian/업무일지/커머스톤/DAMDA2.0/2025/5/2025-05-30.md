- 개발 서버
	- 로그인 후 다음 페이지로 넘어가지 않는 에러가 발생

	> 원인 파악:
	- tableau 서버에서 Timeout 에러가 발생되고 있는 것으로 확인됨

	태블로 서버 핵심 에러:

		1. /opt/tableau/tableau_server/data/tabsvc/logs/httpd
```
error reading status line from remote server localhost:8777
Error reading from remote server returned by /vizql/trusted/
Connection refused: AH00957: http: attempt to connect to 127.0.0.1:8598 (*:80) failed
```
- Apache가 Tableau의 VizQL 서버(localhost:8777)나 다른 내부 서비스(127.0.0.1:8598)에 접근하려 했는데, 해당 포트에서 서비스가 비정상적이거나 죽어 있음.

		2. /app/tableau/data/tabsvc/logs/vizqlserver/vizqlserver_node1-0.log
```
ERROR redis.clients.jedis.JedisFactory - Error while validating pooled Jedis object. redis.clients.jedis.exceptions.JedisConnectionException: Unexpected end of stream.
```
- Redis 연결 실패
	- Redis 연결이 중단되었거나, Redis 서버가 꺼져 있거나 응답하지 않거나, 네트워크 문제 혹은 파라미터 설정 (timeout 등) 문제가 있을 가능성이 높음.

참고 :
	Tableau는 내부적으로 Redis를 캐시용으로 사용
