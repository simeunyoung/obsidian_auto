21) **보안팀은 Amazon EC2 인스턴스에 접근하는 모든 SSH 연결을 감사하고자 합니다. 어떤 조합이 가장 적절합니까?**

		A) AWS Config와 IAM 정책
		B) AWS CloudTrail과 VPC 흐름 로그
		C) AWS Shield와 Security Group
		D) AWS WAF와 ELB 로깅
정답: B

22) **Amazon S3에 저장된 데이터는 주기적으로만 접근됩니다. 기업은 장기 보관 및 비용 절감을 원합니다. 어떤 스토리지가 가장 적절합니까?**

		A) S3 Standard
		B) S3 Intelligent-Tiering
		C) S3 Glacier Deep Archive
		D) S3 Standard-IA
