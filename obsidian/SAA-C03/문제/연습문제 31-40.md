📘 AWS SAA-C03 시나리오 기반 연습문제 (31~40)

31) **한 소셜 미디어 회사는 사용자 이미지 파일을 Amazon S3에 저장하고 있습니다. 사용자는 업로드 후 즉시 썸네일을 확인해야 합니다. 처리 지연을 최소화하면서 자동으로 썸네일을 생성하려면 어떻게 해야 합니까?**

		A) Amazon Rekognition 사용
		B) S3 이벤트를 트리거로 Lambda 함수 사용
		C) EC2 인스턴스에 주기적 작업 등록
		D) CloudWatch 이벤트로 EC2 호출
정답: B

---

32) **한 기업은 AWS에서 운영 중인 웹 애플리케이션을 외부 공격으로부터 보호하고자 합니다. SQL Injection 및 크로스사이트 스크립팅 공격을 방지할 수 있는 서비스는 무엇입니까?**

		A) AWS Shield Standard
		B) Amazon GuardDuty
		C) AWS WAF
		D) AWS Firewall Manager
정답: C

---

33) **개발팀은 Amazon S3 버킷에 중요한 파일을 업로드합니다. 실수로 인한 삭제를 방지하고, 삭제된 경우에도 복원할 수 있어야 합니다. 어떤 설정이 가장 적절합니까?**

		A) S3 Cross-Region Replication
		B) S3 Object Lock 활성화
		C) S3 Versioning 활성화
		D) S3 Static Website Hosting 활성화
정답: C

---

34) **한 회사가 AWS에서 고가용성 웹 애플리케이션을 운영하고 있으며, EC2 인스턴스가 중단될 경우 자동으로 복구되길 원합니다. 가장 적절한 솔루션은 무엇입니까?**

		A) CloudWatch 경보를 생성하여 알림
		B) Auto Scaling을 구성하여 새 인스턴스 시작
		C) EC2 인스턴스에 Auto Recovery 설정

36) **보안팀은 모든 AWS 계정의 보안 상태를 중앙에서 관리하고자 합니다. 어떤 서비스가 이를 가능하게 합니까?**

37) **서버리스 애플리케이션에서 데이터 처리를 위해 Amazon DynamoDB와 Amazon S3를 사용하고 있습니다. 이벤트 기반 아키텍처를 위해 적절한 조합은?**
