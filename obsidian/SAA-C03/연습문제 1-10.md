📘 AWS SAA-C03 시나리오 기반 연습문제 (1~10)

1) **한 스타트업은 매주 주말에 프로모션 캠페인을 진행하며, 웹사이트에 높은 트래픽이 발생합니다. 웹 애플리케이션은 EC2 Auto Scaling 그룹에 배포되어 있으며, 새 인스턴스가 생성될 때마다 애플리케이션 초기화에 2분이 걸립니다. 이로 인해 사용자 요청의 지연이 발생하고 있습니다. 이 문제를 완화할 수 있는 가장 적절한 방법은 무엇입니까?**

		A) Auto Scaling 정책에서 최대 인스턴스 수를 증가시킨다.
		B) 예측 기반 스케줄링을 구성하여 인스턴스를 미리 준비한다.
		C) Spot 인스턴스를 대신 사용하여 빠르게 확장한다.
		D) ELB의 헬스 체크 주기를 줄인다.
정답: B

---

2) **한 회사는 여러 AWS 리전에 웹 애플리케이션을 배포하고 있으며, 전 세계 사용자의 트래픽을 가장 가까운 리전으로 라우팅하여 지연 시간을 줄이고자 합니다. 이 목표를 달성할 수 있는 최적의 솔루션은 무엇입니까?**

		A) Route 53 지연 시간 기반 라우팅 사용
		B) CloudFront를 사용한 정적 콘텐츠 제공
		C) VPC 피어링으로 리전 연결
		D) Application Load Balancer 사용
정답: A

---

3) **보안 팀은 민감한 데이터가 저장된 Amazon S3 버킷이 퍼블릭 액세스를 차단하도록 요구합니다. 또한 특정 IAM 역할만 이 버킷에 접근해야 합니다. 이를 가장 안전하게 구현하는 방법은 무엇입니까?**

		A) S3 버킷 ACL을 사용해 권한 설정
		B) S3 퍼블릭 액세스 차단 활성화 + 버킷 정책으로 역할 제한
		C) S3 버킷에 MFA 삭제 활성화
		D) S3 객체 잠금(Object Lock) 기능 활성화
정답: B

---

4) **한 회사는 데이터 분석을 위해 Amazon S3에 대용량 로그 파일을 저장합니다. 분석 작업은 하루 한 번만 실행되며, 나머지 시간에는 파일이 거의 접근되지 않습니다. 스토리지 비용을 절감할 수 있는 최적의 방법은 무엇입니까?**

		A) S3 Standard
		B) S3 Intelligent-Tiering
		C) S3 Standard-IA
		D) S3 Glacier Deep Archive
정답: C

5) **회사 내부에서 사용하는 애플리케이션이 AWS의 EC2 인스턴스와 RDS 데이터베이스를 사용합니다. 보안팀은 인터넷을 통한 접근 없이 온프레미스 네트워크에서만 EC2와 RDS에 접근하길 원합니다. 어떤 아키텍처가 가장 적절합니까?**

		A) 퍼블릭 서브넷에 인스턴스를 배포
		B) NAT Gateway를 사용하여 외부 접근 차단
		C) VPC 피어링을 통해 온프레미스와 연결
		D) Direct Connect 또는 Site-to-Site VPN을 통해 프라이빗 네트워크 구성
정답: D

---

6) **회사에서 Amazon RDS MySQL을 사용하고 있으며, 데이터베이스 백업은 하루에 한 번 수행됩니다. 그러나 특정 운영 시간 동안 데이터 손실이 발생할 수 있습니다. 더 강력한 복구 전략이 필요한 경우, 어떤 조치를 취하는 것이 가장 적절합니까?**

		A) 스냅샷 빈도를 12시간마다로 증가
		B) Read Replica 생성
		C) 백업을 Glacier로 이동
		D) PITR(Point-In-Time Recovery) 기능 활성화
정답: D

---

7) **한 온라인 쇼핑몰은 갑작스러운 트래픽 증가에 대응하기 위해 웹 서버에 대해 Auto Scaling을 설정했습니다. 그러나 확장 속도가 느리다는 피드백이 있습니다. 확장 시 애플리케이션이 빠르게 요청을 처리할 수 있도록 하기 위한 해결책은 무엇입니까?**

		A) Launch Template을 수정해 빠른 인스턴스 유형 사용
		B) 사용자 정의 이미지를 사용해 인스턴스 시작 시간을 줄임
		C) Step Scaling 대신 Manual Scaling으로 전환
		D) Auto Scaling 그룹에 Spot 인스턴스만 사용
정답: B

8) **DevOps 팀은 EC2 인스턴스에서 실행 중인 애플리케이션의 성능 데이터를 수집하고 시각화하고자 합니다. 또한 알림이 필요할 때 알림도 받아야 합니다. 가장 적절한 서비스 조합은?**

		A) CloudTrail + SNS
		B) CloudWatch Logs + CloudWatch Alarms
		C) AWS Config + CloudWatch
		D) Amazon Inspector + S3

9) **한 회사는 여러 팀에서 AWS 리소스를 생성하고 있습니다. 보안팀은 어떤 리소스가 생성되었는지, 누가 생성했는지 감사 로그를 추적하고 싶습니다. 어떤 서비스를 사용해야 합니까?**

		B) AWS Config
		C) AWS CloudTrail

10) **한 개발팀은 테스트 환경에서 비용을 절감하고자 합니다. EC2 인스턴스를 운영 중이며, 예측 가능한 시간대에만 사용합니다. 이럴 때 가장 비용 효율적인 EC2 구매 옵션은?**

		B) Reserved Instances

		D) Scheduled Reserved Instances (또는 스케줄 기반 Auto Scaling)
